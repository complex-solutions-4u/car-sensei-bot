# Політика конфіденційності

CarSensei обробляє дані виключно з метою надання сервісу аналізу автомобільних оголошень.

Під час використання бота можуть оброблятися та зберігатися:

* Telegram ID користувача;
* історія запитів та результати аналізу;
* посилання на оголошення та інші дані, надіслані користувачем;
* технічна інформація, необхідна для забезпечення роботи сервісу.

Для формування аналітичних висновків частина даних може передаватися до зовнішніх сервісів штучного інтелекту, що використовуються CarSensei як технологічні постачальники. Такі дані обробляються виключно для надання функціоналу сервісу.

CarSensei не продає та не передає персональні дані користувачів третім сторонам з маркетинговою метою.

Історія запитів може зберігатися для покращення якості сервісу, забезпечення безперервності роботи та надання користувачеві доступу до попередніх результатів аналізу.

Дані, надіслані користувачем до бота, не слід розглядати як конфіденційну або чутливу інформацію. Користувач самостійно приймає рішення щодо передачі будь-яких даних через сервіс CarSensei та не повинен надсилати інформацію, розголошення якої може завдати шкоди йому або третім особам.

Користувач має право в будь-який момент звернутися з вимогою про видалення своєї історії запитів та пов'язаних даних. Після отримання відповідного запиту дані будуть видалені в розумний строк, якщо їх зберігання не вимагається законодавством або технічними потребами забезпечення безпеки сервісу.

Аналітичні висновки, оцінки та рекомендації CarSensei мають виключно інформаційний характер і формуються автоматизованими алгоритмами. Вони не є гарантією технічного стану, юридичної чистоти, історії експлуатації або ринкової вартості транспортного засобу. Остаточне рішення щодо придбання автомобіля користувач приймає самостійно та на власний ризик.

Використовуючи бота, користувач погоджується з цією Політикою конфіденційності та обробкою даних у межах, необхідних для роботи сервісу.
